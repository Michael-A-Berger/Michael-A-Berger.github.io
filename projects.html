Honsan

A Single Page Application library web service for anime & manga collections. It is made using ES6, Node, React, Mongoose, Express, and Gulp. The ultimate goal is to create a service that can replace the aging RIT Anime Club library system for anime & manga club rentals.

DPS-kun

A chat bot for the Rochester Rhythm Game Club Discord server. It retrieves wiki links, issues rhythm game challenges, and differentiates between average users and server admins. Currently it is being tested for defects.

GDQ Tally

A website that tracks stats related to the Games Done Quick speedrunning events. This website was being developed with Node.js, Request.js, Moment.js, and the Speedrun.com API. Hosting was handled by Heroku. Currently this project is on hold and under consideration for a revamp.

Anime Audio Visualizer

An audio visualizer that I created for my Rich Media Web Application Development class, IGME 330. It plays (mostly) spoiler-free clips from anime series and overlays the videos with audio effects produced by a set of free songs. This project was built using HTML5 video and audio features alongside Javascript's canvas feature.

Sparklegore

A video game I helped create alongside several classmates from my IGME 106 class at Rochester Institute of Technology. It was made using XNA and I was in charge of implementing the player character physics and collision detection. I also helped compile the code together so that it would all work in one program.

Ember Engine

A 2D video game engine I made out of Java's default libraries when I was in high school. The engine supports custom fonts, sprite scaling, and custom global variables. I never made a game with it, but all of the functionality is there and ready for anyone to use.
